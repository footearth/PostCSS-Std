# cssnext

* http://cssnext.io/
* https://github.com/MoOx/postcss-cssnext

```bash
# cssnext 已经为历史项目，若安装过，请先行卸载
$ npm uninstall cssnext [--save[-dev]]
# 配合 postcss-import postcss-url 一起使用
$ npm install [--save[-dev]] \
  postcss \
  postcss-import \
  postcss-url \
  postcss-cssnext \
  postcss-browser-reporter postcss-reporter
```
